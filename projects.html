Azure Firewall Implementation

Project Overview

Implemented Azure Firewall to control inbound and outbound network access as part of an overall network security plan.

Key Components:

  • Virtual network with workload and jump host subnets
  • Virtual machines in each subnet
  • Custom route ensuring all outbound workload traffic uses the firewall
  • Firewall Application rules allowing outbound traffic to www.bing.com
  • Firewall Network rules allowing external DNS server lookups
